56 // The callback function used for parsing a "double-null terminated string".
57 // When called, the 1st parameter passed in will store the newly extracted sub
58 // string, the 2nd parameter is a 32-bit user defined data, this parameter can
59 // be NULL. The parsing will terminate if this function returns zero. To use
60 // the callback, function pointer needs to be passed to "CIni::ParseDNTString".
61 typedef BOOL (CALLBACK *SUBSTRPROC)(LPCTSTR, LPVOID);
